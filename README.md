# Team Warfare

A simple **prototype inspired by Team Fortress 2**, developed in **GameMaker 8.2** only using **GameMaker 8.1** features.

This project was created for the **Technical Skills Evaluation** at  
**EEST N°2 – 3rd year (3º 1ª)**.

---

## Overview

This repository contains an **early-stage and minimal prototype**.  
At its current state, the project does **not** include:

- Team-based gameplay
- Multiplayer / online features
- Advanced weapon systems
- 3D models or animations

Its main purpose is **learning and experimentation**, serving as a base for future development.

---

## Current Status

- Very early development stage
- Basic systems only
- Offline, single-player prototype
- Subject to frequent changes

---

## Planned Features

- Improved map editor (mouse support, lighting, Win32 API usage)
- Online multiplayer
- Weapons system
- Improved texture handling (custom map textures)
- 3D models and animations

---

## Notes

This is an **educational project** and not a finished game.  
Features may be incomplete, unstable, or subject to refactoring.
